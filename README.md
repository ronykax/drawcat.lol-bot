# drawcat.lol bot

discord app to query drawcat.lol's database

## commands

-   `/cat get [uuid]`
-   `/cat search [name]`
-   `/cat delete [uuid]`
-   `/cat user [uuid | email]`
-   `/cat ban [uuid | email]`

## getting started
1. clone this repo
2. create a `.env` file and fill in the values (refer to [`.env.example`](/.env.example))
3. run `bun run .`
